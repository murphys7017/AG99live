## 使用建议

- 先读“当前有效”文档，再按需查阅归档。
- 如果你关心“下一步怎么做”，优先看《后端主导数据边界与执行计划》。
- 如果你关心“当前代码已经做到哪一步”，优先看《V2 当前实现状态与下一步》。
- 如果你关心“动态主轴的大方向和阶段计划”，优先看《主轴重构计划》。
- 归档文档保留历史上下文，允许包含旧目录名、旧阶段判断和已完成计划。
- 如果归档内容与当前代码冲突，以当前文档和代码为准。
